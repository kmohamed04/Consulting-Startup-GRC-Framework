# Remote Work and Device Security Policy

## 1. Purpose
This policy establishes security requirements for employees working remotely and using company or personal devices to access company systems.

## 2. Scope
This policy applies to:

- All remote and hybrid employees  
- Contractors and interns accessing company resources  
- Any device (company-owned or personal) used for work  

## 3. Remote Work Requirements

### 3.1 Secure Network Use
- Employees must connect through secure, trusted networks.  
- Public Wi-Fi must not be used unless connected through a company-approved VPN.  
- Network sharing and hotspots should be password-protected.

### 3.2 Device Requirements
- Devices must use supported operating systems with automatic updates enabled.  
- Full disk encryption must be enabled on all laptops.  
- Screens must auto-lock after 5 minutes of inactivity.  
- Anti-malware protection must be active and updated.

### 3.3 Data Protection
- Company data must be stored only in approved systems (Google Workspace, company apps).  
- Downloading confidential data to personal devices is prohibited.  
- Copying corporate data to personal cloud storage (Dropbox, iCloud, etc.) is not allowed.

### 3.4 Access Controls
- Multi-factor authentication (MFA) is required for all remote access.  
- Sharing accounts or credentials is strictly prohibited.  
- Remote access must follow least-privilege principles.

## 4. Personal Device Use (BYOD)
- Personal devices may only be used if approved by the Security Lead.  
- Devices must comply with security requirements (encryption, MFA, strong passwords).  
- The company may require removal of corporate data from personal devices upon termination.

## 5. Incident Reporting
Employees must immediately report:
- Lost or stolen devices  
- Unauthorized access  
- Phishing attempts  
- Any suspicious remote access activity  

## 6. Roles and Responsibilities

### IT Administrator
- Ensures remote access tools and VPN configurations are secure.  
- Enforces device security requirements.

### Security Lead
- Approves personal device use and performs risk assessments.  
- Investigates remote access incidents.

### Employees
- Follow all remote access requirements.  
- Report device or security issues immediately.

## 7. Enforcement
Violations may result in loss of remote access privileges, disciplinary action, or termination for serious breaches.
