This project builds a security and compliance framework for Cyverra Consulting, a small IT consulting startup with limited staff and limited budget. The goal is to help the company understand risks, protect client data, and create a simple governance structure. The project uses Excel for the risk register and Power BI for dashboards that track risk levels, control status, and compliance. The framework aligns with NIST CSF and CIS Controls.

Business description: Cyverra Consulting, ten to fifteen employees, mix of office and remote work, cloud based document storage, client reports and internal documents as main assets

Project goals: Identify risks, apply basic controls, reduce security gaps, build simple dashboards, support client trust

Deliverables: Risk register in Excel, Power BI risk dashboard, Power BI compliance view, control mapping to NIST CSF, control mapping to CIS Controls, governance policies, compliance checklist
